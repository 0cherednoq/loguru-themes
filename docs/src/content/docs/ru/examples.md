---
title: Примеры
description: Запускаемые скрипты-примеры, по одному на каждый кейс.
---

Запускаемые скрипты лежат в папке `examples/` репозитория, по одному на каждый
кейс. Из корня проекта:

```bash
pip install -e .
python examples/01_quickstart.py
```

| Файл                  | Кейс                                                            |
| --------------------- | --------------------------------------------------------------- |
| `01_quickstart.py`    | Применить тему одной строкой; все уровни.                      |
| `02_customize.py`     | Подправить тему через неизменяемый API `with_*`.               |
| `03_color_scheme.py`  | Нативные теги следуют палитре; отключение и ограничение областью.|

Остальные кейсы — иконки, свои темы, свои sink-и/формат, запись в файл —
разобраны в руководствах этой документации.
